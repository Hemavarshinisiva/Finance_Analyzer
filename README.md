# Finance_Analyzer
This is an interactive Gradio-based tool that leverages Google Gemini AI to extract and analyze financial data from PhonePe transaction PDFs. Simply upload a text-based PhonePe PDF, and the app generates insightful summaries including monthly income and expenses, unnecessary spending, savings percentage, spending trends, and category-wise breakdowns.
